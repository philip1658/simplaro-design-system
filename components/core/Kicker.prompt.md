Nummerierter UPPERCASE-Kicker über jedem Sektionstitel — das Nummerierungssystem der Marke.

```jsx
<Kicker number="02">Was wir für Sie tun</Kicker>
<Kicker tone="onTerra" number="03">Ein konkreter Hebel</Kicker>
<Kicker tone="onDark">Unser Versprechen</Kicker>
```

Text wird automatisch UPPERCASE gesetzt; Gedankenstrich nach der Nummer kommt vom Component.
